# Lab — Build a Mini AI Gateway

## Overview

In this lab you will build a simplified AI gateway in pure Python. You will implement routing, rate limiting, caching, cost calculation, and audit logging. All provider calls are simulated with mock functions so no real API keys are required.

**Time:** ~35 minutes

---

## What You Will Build

A `AIGateway` class that:

1. **Routes** requests to different backend "providers" based on model name prefix
2. **Rate limits** per API key (sliding window, N requests per minute)
3. **Caches** responses using a hash of model + prompt
4. **Calculates cost** based on provider pricing and token counts
5. **Logs** every request with timestamp, key, model, provider, tokens, and cost
6. **Reports** total cost grouped by API key

## Expected Behavior

```python
# Register a mock provider
def mock_anthropic(model, prompt, max_tokens):
    return ("Mock response", 100, 50)  # (text, input_tokens, output_tokens)

gw = AIGateway(rate_limiter=RateLimiter(requests_per_minute=5))
gw.register_provider("anthropic", mock_anthropic)

# First call — goes to provider
r1 = gw.route(GatewayRequest(key="team-a", model="claude-sonnet", prompt="Hello"))
assert r1.provider == "anthropic"
assert r1.cached == False
assert r1.cost_usd > 0

# Second identical call — served from cache
r2 = gw.route(GatewayRequest(key="team-a", model="claude-sonnet", prompt="Hello"))
assert r2.cached == True

# Cost report
costs = gw.get_cost_by_key()
assert "team-a" in costs
```
