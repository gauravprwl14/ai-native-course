# Lab: Build a Faithfulness Checker

## Goal

Implement a three-function hallucination detection module that can be plugged into any RAG pipeline. The module scores answer faithfulness, identifies contradicting sentences, and produces a structured verification report.

## What You'll Build

| Function | Description |
|----------|-------------|
| `check_faithfulness(answer, context)` | Returns a float 0–1 scoring how well the answer is supported by the context |
| `detect_contradictions(answer, context)` | Returns a list of sentences in the answer that contradict or are unsupported by the context |
| `verify_rag_answer(question, answer, retrieved_chunks)` | Aggregates both checks into a structured report: `{"faithful": bool, "score": float, "issues": list[str]}` |

## Acceptance Criteria

- `check_faithfulness("Refunds in 60 days", "Refunds in 30 days")` returns a float between 0.0 and 1.0
- `check_faithfulness` returns a value < 0.5 when the answer clearly contradicts the context
- `detect_contradictions` returns a non-empty list when contradictions exist
- `detect_contradictions` returns an empty list `[]` when the answer is fully supported
- `verify_rag_answer` returns a dict with keys `"faithful"`, `"score"`, and `"issues"`
- `verify_rag_answer` sets `"faithful": False` when `score < 0.8`
- `verify_rag_answer` sets `"faithful": True` when `score >= 0.8`
- All 8 tests pass: `pytest tests/ -v`
