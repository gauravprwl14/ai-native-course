# Lab 24 — Two-Agent Pipeline

In this lab you will implement a minimal multi-agent system:

1. A `BaseAgent` that wraps the Anthropic API with a system prompt
2. A `ResearchAgent` (specialist: gather bullet-point facts)
3. A `WriterAgent` (specialist: turn notes into polished prose)
4. An `Orchestrator` that runs `ResearchAgent` → `WriterAgent` and assembles the result

## What You'll Build

| Class | Description |
|-------|-------------|
| `BaseAgent` | Base class: `name`, `system_prompt`, `run(task) -> str` |
| `ResearchAgent` | Subclass of BaseAgent; system prompt focused on facts |
| `WriterAgent` | Subclass of BaseAgent; system prompt focused on writing |
| `Orchestrator` | Holds both agents; `run(topic) -> dict` pipeline |

## Your Tasks

### Task A: `BaseAgent.run`

- Call `self.client.messages.create` with:
  - `model=MODEL`
  - `max_tokens=512`
  - `system=self.system_prompt`
  - `messages=[{"role": "user", "content": task}]`
- Return `response.content[0].text`

### Task B: `Orchestrator.run`

- Call `self.researcher.run(f"Research: {topic}")`
- Call `self.writer.run(f"Write a paragraph based on these notes:\n{research}")`
- Return `{"topic": topic, "research": research, "article": article}`

## Stretch Goals

1. Add a `ReviewerAgent` that critiques the article and suggests one improvement

- A bullet-point research summary from the ResearchAgent
- A polished one-paragraph article from the WriterAgent
