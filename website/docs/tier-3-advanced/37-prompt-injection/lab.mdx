# Lab: Build an Injection Detector

## Goal

Build a set of pure-string functions that detect and defend against prompt injection attacks. Most functions require no API calls — this lab focuses on the defensive programming layer.

## What You'll Build

| Function | Description |
|----------|-------------|
| `detect_injection_attempt(text)` | Check text against `INJECTION_PATTERNS`. Returns `(bool, list[str])` — detection flag and matched patterns |
| `sanitize_input(user_input)` | Replace injection phrases with `[REMOVED]` using regex substitution |
| `wrap_user_input(user_input)` | Wrap in `<user_input>` XML tags for prompt isolation |
| `test_injection_resilience(system_prompt, injection_attempts)` | Test a system prompt against a list of injection attempts. Returns `{"total": int, "blocked": int, "passed": int}` |

- `detect_injection_attempt("Ignore all previous instructions")` returns `(True, [...])`
