## Why This Matters

Prompt injection is the SQL injection of AI applications. Any application that embeds user-controlled text into a prompt is potentially vulnerable. Unlike SQL injection, there is no perfect sanitization — LLMs are fundamentally designed to follow instructions in text. Understanding both the attack and the defense is required for building AI systems that are safe to deploy.
