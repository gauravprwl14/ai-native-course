## How This Connects

Fine-tuning (Chapter 29) teaches a model to follow instructions by example. But a well-fine-tuned model can still produce harmful, dishonest, or unhelpful outputs — because the training signal was "predict the next token", not "be good". RLHF and DPO are the techniques the industry uses to close this gap.

OpenAI used RLHF to create InstructGPT (the predecessor to ChatGPT). Anthropic uses Constitutional AI, a variant of the same ideas, to train Claude. Every major aligned LLM today was shaped by these methods.

In this chapter you will understand how they work and build the foundational artifact they depend on: a preference dataset.
