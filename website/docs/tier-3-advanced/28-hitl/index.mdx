## Why This Matters

Fully autonomous agents are powerful — and risky. An agent that can send emails, delete files, or make purchases is only useful if it can be trusted. HITL (Human-in-the-Loop) is the design pattern that draws the line between what the agent does autonomously and what requires a human decision.

Every production-grade agent system needs some form of HITL. The question isn't *whether* to add human oversight — it's *where* and *when*. Too many checkpoints and you've defeated the point of automation. Too few and you're one confused LLM away from deleting production data.
