## How This Connects

Every chapter so far has used `client.messages.create(...)` — a blocking call that waits for the full response. That's fine for labs. It's bad for UX: users stare at a blank screen for 5–10 seconds.

Streaming fixes this by sending tokens as they're generated. The model starts responding in milliseconds. This chapter teaches you how to implement streaming end-to-end: from the Anthropic API to a Python generator to a web endpoint.
