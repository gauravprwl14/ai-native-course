## How This Connects

Chapter 19 showed you how an LLM can call a single tool and return a result. That's powerful, but real-world tasks rarely fit in one step. "Search the web, summarize the top 3 results, then write a report" requires multiple rounds of tool use — each step informed by the result of the last.

The agentic loop is the architectural pattern that makes this possible. The model iterates: it thinks about what to do, takes an action (tool call), observes the result, then thinks again. It repeats until the task is done or a safety limit is hit. This is the foundation of every autonomous AI agent.
