## How This Connects

RAG is the most widely deployed pattern in production AI systems. It bridges the gap between static LLM training data and the dynamic, private knowledge that organisations actually need to query. Embeddings (Chapter 05) power the retrieval step. System prompts (Chapter 11) control how the LLM uses retrieved context. Vector databases (Chapter 15) scale RAG to millions of documents. Advanced RAG variants add chunking strategies, re-ranking, and hybrid search on top of this core pattern.
