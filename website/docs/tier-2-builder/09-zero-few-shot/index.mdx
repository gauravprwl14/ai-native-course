# Chapter 09 — Zero/Few-shot Prompting

## What You'll Learn

- The difference between zero-shot and few-shot prompting
- When to use examples in your prompts (and when not to)
- How many examples to use and how to pick them
- What label balance means and why it matters
- How in-context learning works without updating model weights

## Prerequisites

- Tier 1 complete — you can make API calls, understand tokens and context windows
- Python 3.10+, `anthropic` SDK installed
