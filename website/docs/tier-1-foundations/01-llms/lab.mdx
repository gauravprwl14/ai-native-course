# Lab 01 — Your First LLM API Call

## What You're Building

A Python script that:
1. Calls Claude via the Anthropic API
2. Experiments with temperature (0 vs 0.7 vs 1.0) on the same prompt
3. Compares two models on the same task
4. Estimates the cost of each call

## Prerequisites

```bash
cd curriculum/shared
pip install -r requirements.txt
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
```

Get a free API key at [console.anthropic.com](https://console.anthropic.com).

## Acceptance Criteria

- [ ] Script runs without errors
- [ ] Makes at least 3 API calls (different temperatures)
- [ ] Prints responses and shows token count + cost estimate
- [ ] Tests pass: `pytest tests/ -v`

## Key Concepts You'll Use

- `anthropic.Anthropic()` — creates the client
- `client.messages.create()` — makes an API call
- `response.content[0].text` — gets the text response
- `response.usage.input_tokens` + `response.usage.output_tokens` — token counts
