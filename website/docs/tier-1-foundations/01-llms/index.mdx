# Chapter 01 — LLMs & How They Work

## What You'll Learn
- What a large language model (LLM) is and how it generates text
- The difference between GPT, Claude, Gemini, and Llama
- How to make your first API call to Claude
- What "tokens", "temperature", and "context window" mean at a high level

## Prerequisites
- Python installed (3.10+)
- pip / virtualenv basics
- A terminal you're comfortable with
