# Lab 04 — Compare Outputs at Different Temperatures

## What You're Building

A Python module that calls the Anthropic API at multiple temperature settings and surfaces the differences. You'll implement three functions:

1. **`generate_at_temperature`** — generate `n` responses from a prompt at a given temperature
2. **`compare_temperatures`** — run the same prompt across a list of temperatures, returning all responses
3. **`is_valid_json_output`** — validate whether a string is valid JSON (useful for testing structured outputs)

## Acceptance Criteria

- [ ] `generate_at_temperature(prompt, 0.0, n=3)` returns a list of exactly 3 strings
- [ ] `generate_at_temperature` passes the correct temperature to the API
- [ ] `compare_temperatures` returns a dict with one key per temperature value
- [ ] Each dict value is a list of `n` response strings
- [ ] `is_valid_json_output('{"key": "value"}')` returns `True`
- [ ] `is_valid_json_output("not json")` returns `False`
- [ ] All tests pass: `pytest tests/ -v`

## Stretch Goals

Once the core functions work, try these extensions:

1. **Uniqueness score** — for a given prompt and temperature, what fraction of the `n` responses are distinct?
2. **JSON success rate** — generate 10 responses asking for JSON output at temperature=0.0 vs temperature=1.0. What fraction is valid JSON at each setting?
3. **Longest common substring** — measure how similar responses are to each other at different temperatures.
