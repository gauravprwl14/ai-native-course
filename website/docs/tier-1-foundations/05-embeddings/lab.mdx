# Lab 05 — Build Semantic Search with Embeddings

## What You're Building

A Python module that embeds text and finds the most semantically similar string from a list of candidates — the core of any semantic search system.

You will implement three functions:

1. **`embed_text(text)`** — call the OpenAI embeddings API and return the vector
2. **`cosine_similarity(v1, v2)`** — compute the cosine similarity between two vectors from scratch
3. **`find_most_similar(query, candidates)`** — find the most semantically similar candidate for a given query

## Acceptance Criteria

- [ ] `embed_text(text)` returns a `list[float]` with 1536 elements
- [ ] `embed_text` calls the OpenAI API with `model="text-embedding-3-small"`
- [ ] `cosine_similarity([1,0], [1,0])` returns `1.0`
- [ ] `cosine_similarity([1,0], [0,1])` returns `0.0`
- [ ] `cosine_similarity([1,0], [-1,0])` returns `-1.0`
- [ ] `find_most_similar("I love dogs", ["cats are great", "I enjoy dogs", "pizza is tasty"])` returns `("I enjoy dogs", score)` where score > 0.8
- [ ] `find_most_similar` returns a `tuple[str, float]`
- [ ] All tests pass: `pytest tests/ -v`
